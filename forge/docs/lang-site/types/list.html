list built-in type

Generic ordered collection. Declared as `list<T>` where T is the element type.

list<T>.length -> int

Get the number of elements in the list. Accessed as a property, not a method call.

[1, 2, 3].length  // => 3

list<T>.push(val: T) -> list<T>

Append an element to the list, returning the new list.

[1, 2].push(3)  // => [1, 2, 3]

list<T>.map(fn: (T) -> U) -> list<U>

Transform each element using a function, returning a new list.

[1, 2, 3].map((x) -> x * 2)  // => [2, 4, 6]

list<T>.filter(fn: (T) -> bool) -> list<T>

Keep only elements for which the function returns true.

[1, 2, 3, 4].filter((x) -> x > 2)  // => [3, 4]

list<T>.sorted() -> list<T>

Return a new list with elements sorted in ascending order.

[3, 1, 2].sorted()  // => [1, 2, 3]

list<T>.each(fn: (T) -> void)

Iterate over elements for side effects. Returns void.

[1, 2, 3].each((x) -> println(string(x)))

list<T>.find(fn: (T) -> bool) -> T?

Return the first element matching the predicate, or null.

[1, 2, 3].find((x) -> x > 1)  // => 2

list<T>.any(fn: (T) -> bool) -> bool

Return true if any element matches the predicate.

[1, 2, 3].any((x) -> x > 2)  // => true

list<T>.all(fn: (T) -> bool) -> bool

Return true if all elements match the predicate.

[1, 2, 3].all((x) -> x > 0)  // => true

list<int>.sum() -> int

Sum all elements in a list of integers.

[1, 2, 3].sum()  // => 6

list<string>.join(sep: string) -> string

Join all elements into a single string with a separator.

["a", "b", "c"].join(", ")  // => "a, b, c"

list<T>.reduce(fn: (acc: T, val: T) -> T) -> T

Reduce the list to a single value by applying a function cumulatively.

[1, 2, 3].reduce((acc, x) -> acc + x)  // => 6

list<T>.enumerate() -> list<{index: int, value: T}>

Return a list of index-value pairs.

["a", "b"].enumerate()  // => [{index: 0, value: "a"}, ...]

list<T>.clone() -> list<T>

Create a shallow copy of the list.

let copy = items.clone()
