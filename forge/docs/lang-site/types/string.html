string built-in type

UTF-8 string type. Immutable value type with a rich set of built-in methods.

Methods

MethodDescription
string.length() -> intGet the number of characters in the string.
string.split(sep: string) -> list<string>Split the string by a separator, returning a list of parts.
string.trim() -> stringRemove leading and trailing whitespace.
string.contains(sub: string) -> boolCheck if the string contains a substring.
string.upper() -> stringConvert all characters to uppercase.
string.lower() -> stringConvert all characters to lowercase.
string.starts_with(prefix: string) -> boolCheck if the string starts with a given prefix.
string.ends_with(suffix: string) -> boolCheck if the string ends with a given suffix.
string.replace(old: string, new: string) -> stringReplace all occurrences of a substring with another.
string.parse_int() -> intParse the string as an integer.
string.repeat(n: int) -> stringRepeat the string n times.

string.length() -> int

Get the number of characters in the string.

"hello".length()  // => 5

string.split(sep: string) -> list<string>

Split the string by a separator, returning a list of parts.

"a,b,c".split(",")  // => ["a", "b", "c"]

string.trim() -> string

Remove leading and trailing whitespace.

"  hello  ".trim()  // => "hello"

string.contains(sub: string) -> bool

Check if the string contains a substring.

"hello world".contains("world")  // => true

string.upper() -> string

Convert all characters to uppercase.

"hello".upper()  // => "HELLO"

string.lower() -> string

Convert all characters to lowercase.

"HELLO".lower()  // => "hello"

string.starts_with(prefix: string) -> bool

Check if the string starts with a given prefix.

"hello".starts_with("hel")  // => true

string.ends_with(suffix: string) -> bool

Check if the string ends with a given suffix.

"hello".ends_with("llo")  // => true

string.replace(old: string, new: string) -> string

Replace all occurrences of a substring with another.

"hello".replace("l", "r")  // => "herro"

string.parse_int() -> int

Parse the string as an integer.

"42".parse_int()  // => 42

string.repeat(n: int) -> string

Repeat the string n times.

"ab".repeat(3)  // => "ababab"
