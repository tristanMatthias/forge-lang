With Expression stable

expr with { field: val } — immutable copy with overrides

Struct update syntax: expr with { field: value } creates a copy with updated fields

The `with` expression creates a modified copy of an immutable struct. Given `let p = Point { x: 1, y: 2 }`, writing `let q = p with { x: 10 }` creates a new Point where `x` is 10 and `y` is copied from `p`. The original `p` is unchanged.

This is the idiomatic way to "update" immutable data in Forge. Rather than mutating fields in place, you express transformations as new values derived from old ones. This makes data flow explicit and eliminates bugs caused by unexpected mutation of shared references.

The `with` expression copies all fields from the original, then applies the overrides. Only the fields you specify are different; everything else is preserved. This is concise even for structs with many fields, since you only mention what changes.

This feature is equivalent to the spread/rest operator for objects in JavaScript (`{ ...obj, field: newValue }`), Kotlin's `copy()` method on data classes, or Rust's struct update syntax (`Point { x: 10, ..p }`). Forge's `with` keyword reads naturally in English, making the intent immediately clear.

Syntax

expr with { field: value }

Examples

composition_with_structs.fg

/// # Immutable Updates with 'with'
///
/// Creating modified copies of structs without
/// mutation, enabling functional data patterns.
///

type Person = { name: string, age: int }

fn birthday(p: Person) -> Person {
  p with { age: p.age + 1 }
}

fn main() {
  let alice = Person { name: "Alice", age: 30 }
  let older = birthday(alice)
  println(older.name + " is " + string(older.age))
  println(alice.name + " is " + string(alice.age))
}

nested_update.fg

/// # With Expression
///
/// Create updated copies of structs using `with`.
///

type Point = { x: float, y: float }

fn main() {
  let p = Point { x: 3.0, y: 4.0 }
  let moved = p with { x: 10.0 }
  println(string(moved.x))
  println(string(moved.y))
}

struct_update.fg

/// # Struct Update
///
/// Create a modified copy of a struct with `with`.
///

type Person = { name: string, age: int }

fn main() {
  let p = Person { name: "alice", age: 25 }
  let p2 = p with { age: 30 }
  println(p2.name)
  println(p2.age)
}

with_chained.fg

/// # With Chained
///
/// Apply `with` in sequence to update fields step by step.
///

type Person = { name: string, age: int }

fn main() {
  let p = Person { name: "alice", age: 25 }
  let p2 = p with { name: "bob" }
  let p3 = p2 with { age: 30 }
  println(p3.name)
  println(string(p3.age))
}

with_computed.fg

/// Test: with expression using computed field values (slug pattern)
///

type Post = { title: string, slug: string }

fn main() {
  let data = Post { title: "Hello World", slug: "" }
  let updated = data with { slug: data.title.lower().replace(" ", "-") }
  println(updated.slug)
  println(updated.title)
}

with_computed_value.fg

/// # With Computed Value
///
/// Field values in `with` expressions can be computed from the original.
///

type Person = { name: string, age: int }

fn main() {
  let p = Person { name: "alice", age: 25 }
  let older = p with { age: p.age + 1 }
  println(older.name)
  println(string(older.age))
}

with_multiple.fg

/// Test: With expression updating multiple fields at once
///

type User = { name: string, email: string, age: int }

fn main() {
  let user = User { name: "alice", email: "alice@old.com", age: 25 }
  let updated = user with { name: "bob", email: "bob@new.com", age: 30 }
  println(updated.name)
  println(updated.email)
  println(string(updated.age))
}

with_multiple_fields.fg

/// # With Multiple Fields
///
/// Update several fields of a struct at once using `with`.
///

type User = { name: string, email: string, age: int }

fn main() {
  let u = User { name: "alice", email: "alice@test.com", age: 25 }
  let u2 = u with { name: "charlie", email: "charlie@example.com", age: 35 }
  println(u2.name)
  println(u2.email)
  println(string(u2.age))
  println(u.name)
}

with_nested_struct.fg

/// # With Nested Struct
///
/// Using `with` on a struct that contains other struct values.
///

type Config = { label: string, value: int }

fn main() {
  let c = Config { label: "original", value: 42 }
  let c2 = c with { label: "updated", value: 100 }
  println(c2.label)
  println(string(c2.value))
}
