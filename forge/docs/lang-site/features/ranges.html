Ranges stable

start..end (exclusive), start..=end (inclusive) — range expressions

Range expressions: 0..10 (exclusive) and 0..=10 (inclusive)

Ranges represent a sequence of consecutive values, written with the `..` operator. An exclusive range `0..5` includes 0, 1, 2, 3, 4. An inclusive range `0..=5` also includes 5. Ranges are most commonly used in `for` loops: `for i in 0..n { ... }`.

Ranges are first-class values that can be stored in variables and passed to functions. They support the `contains` method for membership testing: `(1..10).contains(5)` returns true. This makes ranges useful for validation and bounds checking beyond just iteration.

The exclusive range `..` is the default because it aligns with zero-based indexing. When you write `for i in 0..list.length() { ... }`, there is no off-by-one risk. Use the inclusive form `..=` when you specifically need the endpoint, such as `for day in 1..=31 { ... }`.

Syntax

start..end
start..=end

Examples

basic_range.fg

/// # Basic Range
///
/// Exclusive range `0..5` and inclusive range `0..=5`.
///

fn main() {
  mut sum = 0
  for i in 0..5 {
    sum = sum + i
  }
  println(sum)
}

inclusive_range.fg

/// # Inclusive Range
///
/// `..=` creates an inclusive range.
///

fn main() {
  mut sum = 0
  for i in 1..=5 {
    sum = sum + i
  }
  println(sum)
}

range_in_list.fg

/// # Range as List Index
///
/// Use a range to iterate over a slice of list indices.
///

fn main() {
  let items = [10, 20, 30, 40, 50]
  for i in 1..4 {
    println(items[i])
  }
}

range_in_loop.fg

/// # Range in Loop
///
/// Use ranges to drive for loops with inclusive and exclusive bounds.
///

fn main() {
  mut sum = 0
  for i in 0..5 {
    sum = sum + i
  }
  println(string(sum))

  mut sum2 = 0
  for i in 0..=5 {
    sum2 = sum2 + i
  }
  println(string(sum2))
}

range_reverse.fg

/// # Reverse Range
///
/// Adversarial test: range where start > end should iterate zero times.
///

fn main() {
  mut count = 0
  for i in 10..5 {
    count = count + 1
  }
  println(`count: ${count}`)
}

range_single.fg

/// # Single Element Range
///
/// A range of one element (n..n+1) iterates exactly once.
/// An empty range (n..n) iterates zero times.
///

fn main() {
  for i in 5..6 {
    println(i)
  }

  mut count = 0
  for i in 5..5 {
    count = count + 1
  }
  println(`empty range count: ${count}`)
}

range_variables.fg

/// # Range with Variable Bounds
///
/// Range start and end can be expressions or variables.
///

fn main() {
  let start = 2
  let end = 5
  mut sum = 0
  for i in start..end {
    sum = sum + i
  }
  println(`sum: ${sum}`)
}

range_zero_zero.fg

/// # Zero to Zero Range
///
/// Adversarial test: 0..0 should be empty, 0..=0 should have one element.
///

fn main() {
  mut count = 0
  for i in 0..0 {
    count = count + 1
  }
  println(`exclusive 0..0: ${count}`)

  mut count2 = 0
  for i in 0..=0 {
    count2 = count2 + 1
  }
  println(`inclusive 0..=0: ${count2}`)
}
