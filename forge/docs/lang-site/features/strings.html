Strings stable

UTF-8 strings with built-in methods

String methods: length, upper, lower, trim, contains, split, starts_with, ends_with, replace, parse_int, repeat

Strings in Forge are UTF-8 encoded, immutable sequences of characters. String literals use double quotes: `"hello world"`. Template literals with `${}` interpolation provide the primary way to build strings dynamically: `"Hello, ${name}! You are ${age} years old."`.

Strings support a comprehensive set of methods: `length()`, `contains(sub)`, `starts_with(prefix)`, `ends_with(suffix)`, `to_upper()`, `to_lower()`, `trim()`, `split(separator)`, `replace(old, new)`, `substring(start, end)`, and more. These methods return new strings rather than mutating in place, consistent with Forge's immutability-first design.

String comparison uses `==` for value equality, not reference equality. Strings can be concatenated with `+`, though template literals are preferred for building complex strings since they are more readable and less error-prone than chained concatenation.

Multi-line strings are supported naturally. Forge does not have a separate character type; single characters are simply strings of length one.

Syntax

"hello"
s.length()
s.split(sep)

Examples

empty_string_ops.fg

/// Test: Operations on empty strings
///

fn main() {
  let empty = ""
  println(string(empty.length))
  println(empty.upper())
  println(empty.lower())
  println(string(empty.contains("")))
  println(string(empty.starts_with("")))
  println(string(empty.ends_with("")))
  let parts = empty.split(",")
  println(string(parts.length))
}

string_as_fn_return.fg

/// Test: String operations in function return values
///

fn shout(s: string) -> string {
  s.upper()
}

fn first_word(s: string) -> string {
  s.split(" ")[0]
}

fn middle(s: string) -> string {
  s.replace("hell", "").replace("ld", "")
}

fn main() {
  println(shout("hello world"))
  println(first_word("hello world"))
  println(middle("hello world"))
}

/// Test: Chained string methods for slug generation
///

fn main() {
  let title = "Hello World"
  let slug = title.lower().replace(" ", "-")
  println(slug)

  let upper = "hello world".upper()
  println(upper)

  let email = "alice@test.com"
  println(string(email.contains("@")))
  println(string(email.starts_with("bob")))
}

/// Test: String methods — trim, upper, lower, contains, starts_with, replace

}
