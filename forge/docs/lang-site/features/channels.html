Channels stable

ch <- val (send), <- ch (recv) — typed channel communication

Channel-based communication with send (<-), receive (<-), and iteration

Channels are typed conduits for communication between concurrent tasks. Create a channel with `channel.new()`, send values with `ch <- value`, and receive with `<- ch`. Channels are the safe, structured way to pass data between spawned tasks without shared mutable state.

Channels are generic: `channel.new<int>()` creates a channel that carries integers. The type system ensures you never accidentally send a string through an int channel. Both the send and receive operations are type-checked at compile time.

Channels can be iterated with `for msg in ch { ... }`, which receives messages in a loop until the channel is closed with `channel.close(ch)`. This pattern is the idiomatic way to process a stream of values from a producer task. Timed channels created with `channel.tick(ms)` send a value at regular intervals, useful for periodic tasks.

Forge channels follow the same model as Go channels. They are unbuffered by default, meaning a send blocks until a receiver is ready. This synchronization property makes channel-based programs easier to reason about than lock-based alternatives.

Syntax

ch <- value
<- ch
for msg in ch { }

Examples

basic_channel.fg

/// # Basic Channel
///
/// Create a buffered channel, send a value, and receive it.
///

use @std.channel

fn main() {
  let ch = channel(1)
  ch <- "hello"
  let msg = <- ch
  println(msg)
}

buffered_channel.fg

/// # Buffered Channel
///
/// Create a buffered channel and check its properties.
///

use @std.channel

fn main() {
  let ch = channel(3)

  ch <- "1"
  ch <- "2"
  ch <- "3"

  println(string(ch.length))
  println(string(ch.capacity))
  println(string(ch.is_full))

  let a = <- ch
  let b = <- ch
  println(a)
  println(b)
  println(string(ch.length))
  println(string(ch.is_full))
}

channel_iteration.fg

/// # Channel Iteration
///
/// Iterate over a channel with a for loop and tick timer.
///

use @std.channel

fn main() {
  let tick = channel.tick(50)

  mut count = 0
  for _ in tick {
    count = count + 1
    if count >= 3 { break }
  }

  println(string(count))
}

composition_with_spawn.fg

/// # Producer-Consumer with Channels
///
/// Spawning concurrent tasks that communicate
/// through channels with iteration.
///

use @std.channel

fn main() {
  let ch = channel(3)

  spawn {
    for i in 0..3 {
      ch <- i
    }
    channel.close(ch)
  }

  sleep(50)
  for msg in ch {
    println("received: " + string(msg))
  }
}

spawn_channel.fg

/// # Spawn with Channel
///
/// Send data between spawned goroutines via channels.
///

use @std.channel

fn main() {
  let ch = channel(1)

  spawn {
    ch <- "hello from spawn"
  }

  sleep(50)
  let msg = <- ch
  println(msg)
}

typed_channel.fg

/// # Typed Channel
///
/// Create a typed channel with `channel<string>(N)` syntax.
/// The type parameter enables compile-time type checking of send/receive.
///

use @std.channel

fn main() {
  let ch = channel<string>(1)
  ch <- "hello typed"
  let msg = <- ch
  println(msg)
}
