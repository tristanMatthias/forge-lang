Defer stable

defer expr — execute when scope exits, LIFO order

Deferred execution of expressions before function return, executed in reverse order

The `defer` statement schedules an expression to execute when the enclosing scope exits, regardless of whether the exit is normal or due to an error. This is the primary mechanism for resource cleanup in Forge: `let f = open(path); defer close(f)`. No matter how the function returns, the file will be closed.

Deferred expressions execute in LIFO (last-in, first-out) order. If you defer A then defer B, B executes first, then A. This matches the natural pattern of resource acquisition: resources acquired later should be released first.

Defer eliminates the need for finally blocks, destructors, or RAII patterns for resource management. It keeps the cleanup code next to the acquisition code, rather than at the bottom of a try/finally block potentially hundreds of lines away. This locality makes it easy to verify that every resource is properly cleaned up.

The concept comes from Go, where `defer` is used extensively. Forge's implementation works the same way, executing deferred expressions before every return point in the function, including early returns and error propagation with `?`.

Syntax

defer expr

Examples

basic_defer.fg

/// # Defer
///
/// `defer` runs an expression when the function returns.
///

fn main() {
  defer println("cleanup")
  println("hello")
}

composition_with_closures.fg

/// # Resource Cleanup with Defer
///
/// Using defer for guaranteed cleanup in
/// functions that return values.
///

fn compute() -> int {
  defer println("cleanup complete")
  println("working...")
  42
}

fn main() {
  let result = compute()
  println("result: " + string(result))
}

defer_cleanup.fg

/// Test: defer for cleanup pattern (resource management)
///

fn main() {
  defer println("cleanup a")
  defer println("cleanup b")
  println("working")
}

defer_conditional_return.fg

/// # Defer with Conditional No-Return Path
///
/// Defer runs when function exits through the normal path.
///

fn maybe_return(early: bool) {
  defer println("always runs")
  if early {
    println("early path")
    return
  }
  println("late path")
}

fn main() {
  maybe_return(false)
}

defer_early_return_value.fg

/// # Defer with Early Return Value
///
/// Defer runs before function returns a value.
///

fn compute() -> int {
  defer println("cleanup")
  return 42
}

fn main() {
  let result = compute()
  println(string(result))
}

defer_in_loop.fg

/// # Defer in Loop
///
/// Defer inside a loop body registers at function scope.
/// The deferred expression runs once when the function exits.
///

fn main() {
  defer println("main defer")
  for i in 0..3 {
    defer println("loop defer")
    println(`body ${i}`)
  }
  println("after loop")
}

defer_many.fg

/// # Many Defers
///
/// Five defers in reverse order.
///

fn main() {
  defer println("1")
  defer println("2")
  defer println("3")
  defer println("4")
  defer println("5")
  println("work")
}

defer_multiple_functions.fg

/// # Defer in Multiple Functions
///
/// Each function has its own defer stack.
///

fn func_b() {
  defer println("b-defer")
  println("b-start")
}

fn func_a() {
  defer println("a-defer")
  println("a-start")
  func_b()
}

fn main() {
  func_a()
}

defer_nested_functions.fg

/// # Defer in Nested Functions
///
/// Each function's defers run when that function returns,
/// in reverse order within the function.
///

fn inner() {
  defer println("inner defer")
  println("inner start")
}

fn outer() {
  defer println("outer defer")
  inner()
}

fn main() {
  outer()
}

defer_return_in_if.fg

/// # Defer Fires on Return Inside If
///
/// Defer must execute even when return is inside an if block.
///

fn test(x: int) -> string {
  defer println("cleanup")
  if x > 0 {
    return "positive"
  }
  return "non-positive"
}

fn main() {
  println(test(5))
}

defer_with_if.fg

/// # Defer with Conditional
///
/// Defer inside an if block still runs at function exit.
///

fn main() {
  let x = true
  if x {
    defer println("conditional defer")
  }
  println("work")
}

defer_with_return.fg

/// # Defer with Return
///
/// Deferred statements run before an explicit return.
///

fn early_return() {
  defer println("deferred")
  println("inside")
  return
}

fn main() {
  early_return()
}

defer_with_string_template.fg

/// # Defer with String Template
///
/// Defer captures variable by reference — sees final value in template string.
///

fn main() {
  mut count = 0
  defer println(`count: ${count}`)
  for i in 0..10 {
    count = count + 1
  }
  println("done")
}

defer_with_variables.fg

/// # Defer with Variables
///
/// Defer captures the variable reference, not the value at declaration time.
/// The deferred println sees the final value of the variable.
///

mut x = 0

fn main() {
  defer println(`x is ${x}`)
  x = 42
  println("working")
}
