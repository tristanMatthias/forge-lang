Collections stable

list and map literals with type-safe operations

List and map literal expressions with type inference

Forge provides two built-in collection types: `list<T>` and `map<K, V>`. Lists are ordered, indexed sequences created with square brackets: `let nums = [1, 2, 3]`. Maps are key-value stores created with curly braces: `let ages = { "Alice": 30, "Bob": 25 }`.

Lists support a rich set of methods: `push`, `pop`, `map`, `filter`, `reduce`, `each`, `length`, `contains`, `sorted`, `join`, and more. Chaining these methods with closures is the idiomatic way to transform data: `numbers.filter((n) -> n > 0).map((n) -> n * 2)`. Lists are generic, so `list<int>`, `list<string>`, and `list<list<int>>` are all valid types.

Maps support `get`, `set`, `keys`, `values`, `contains`, and `length`. Map access uses bracket notation: `ages["Alice"]`. When a key might not exist, use the null-safe access `ages["Charlie"]?` combined with `??` to provide a default.

Both collections are mutable when declared with `mut`. Immutable collections cannot have elements added or removed, making them safe to share across function boundaries without defensive copying.

Syntax

[1, 2, 3]
{ key: value }

Examples

list_any_all.fg

/// # List Any and All
///
/// Check if any/all elements satisfy a predicate.
///

fn main() {
  let nums = [2, 4, 6, 8]
  println(string(nums.any((x) -> x > 5)))
  println(string(nums.any((x) -> x > 10)))
  println(string(nums.all((x) -> x > 0)))
}

/// # Basic List Operations
///
/// Create a list, check length, access elements.

fn main() {
  let nums = [10, 20, 30]
  println(string(nums.length))
  println(string(nums[0]))
  println(string(nums[2]))
}

list_chain_operations.fg

/// # Chained List Operations
///
/// Chain multiple list operations: filter, map, reduce.
///

fn main() {
  let nums = [1, 2, 3, 4, 5, 6]
  let result = nums
    .filter((x) -> x > 2)
    .map((x) -> x * 2)
    .reduce(0, (acc, x) -> acc + x)
  println(string(result))
}

/// # List Find
///
/// Find the first element matching a predicate.
///

fn main() {
  let nums = [10, 20, 30, 40, 50]
  let found = nums.find((x) -> x > 25)
  if found != null {
    println("found")
    let val = found!
    println(string(val))
  }
}

list_map_filter.fg

/// # List Map and Filter
///
/// Chain map and filter on a list.
///

fn main() {
  let nums = [1, 2, 3, 4, 5]
  let doubled = nums.map((x) -> x * 2)
  let big = doubled.filter((x) -> x > 3)
  big.each((x) -> println(string(x)))
}

/// # List of Strings
///
/// Operations on a list of strings.
///

fn main() {
  let words = ["hello", "world", "foo"]
  println(string(words.length))
  let upper = words.map((w) -> w.upper())
  println(upper[0])
  println(words.join("-"))
}

///

/// # Basic Map Operations
///
/// Create a map, get values (nullable), get keys.
///

fn main() {
  let m = {"a": 42, "b": 99}
  let val = m.get("a")
  if val != null {
    let v = val!
    println(string(v))
  }
  let missing = m.get("c")
  if missing == null {
    println("null")
  }
  let k = m.keys()
  println(string(k.length))
}
