# Forge Principles:

- LLMs are the primary writers of code, and we should design for them first
- Futuristic wherever possible
- Collapse the paradox: EG: Safe + Easy with scope abstractions  
- Convention over configuration, with escape hatches where you want them
- Plays nice with others (BYO front end, bind around rust/go/etc)
- Beautiful
  - Really clean DX, exceptional errors docs and type system, dramatically reduced boilerplate
  - Validation errors from server should be so good they're pipeable straight to the UI
- Performant: Compiled, safe, fast
